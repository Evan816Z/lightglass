# LightGlass UI 原型设计说明

> 整体风格: **Windows 11 Fluent Design + Apple VisionOS 玻璃质感**, 深色优先。

---

## 1. 设计系统

### 1.1 调色板

| 名称 | 颜色 | 用途 |
|------|------|------|
| Ink 950 | `#06080F` | 整体深色背景 |
| Ink 900 | `#0B0F1A` | 主背景 |
| Ink 800 | `#101523` | 卡片底色 |
| Ink 700 | `#181E2D` | 次级表面 |
| Ink 400 | `#6B7793` | 弱化文本 |
| Ink 100 | `#E6EAF2` | 主文本 |
| Violet | `#7C5CFF` | 主色, 强调 |
| Mint | `#36E0C7` | 次色, 状态 |
| Rose | `#FF7AB6` | 警告 / 删除 |
| Amber | `#FFB454` | 提示 |

### 1.2 字体

- 标题: `Sora` 600/700
- 正文: `Inter` 400/500/600
- 数据: `JetBrains Mono`

### 1.3 间距 / 圆角 / 阴影

- 间距: 4 / 8 / 12 / 16 / 24 / 32
- 圆角: 8 (小) / 12 (中) / 16 (大) / 24 (窗口)
- 阴影: 玻璃感 24px blur + 1px 内描边

---

## 2. 页面原型

### 2.1 登录页 (`/login`)

- 左侧品牌插画: 渐变光斑 + Sora 大标题 + 三特性胶囊
- 右侧: 居中毛玻璃卡片, 含 mode 切换 (登录 / 注册)、邮箱、密码、提交按钮
- 错误: 内嵌红框提示

### 2.2 控制台 (`/console`)

- 顶部: 玻璃导航条 (Logo / 用户 / 退出)
- 主体: 项目卡片网格, 每张卡片含缩略图 (渐变 + 网格背景 + 项目名) + 操作 (查看 / 删除)
- 右上: 「新建项目」主按钮
- 新建弹窗: 名称 + slug (实时联动)

### 2.3 编辑器 (`/editor/:projectId`)

#### 顶部工具栏 (48px, 玻璃)
- 左: 返回按钮 + 项目名 + slug
- 中: 撤销 / 重做 / 辅助线 / 吸附 / 网格 / JSON / 预览
- 右: 保存
- 中间状态: ● 保存中 / 已保存 1 分钟前

#### 左侧组件库 (256px, 玻璃)
- 顶部 Tab: 组件 / 图层
- 组件: 文字 / 图片 / 视频 / 音频 / Web (图标 + 文字, 2 列)
- 模板: 欢迎桌面 (示例)
- 点击添加即在画布中央偏左添加

#### 中央画布
- 桌面背景: 用户配置的渐变 / 纯色 / 图片
- 画布尺寸: 1280 × 800 (可改)
- 自动缩放以适应可用空间
- 窗口堆叠: 选中时显示紫色 outline + 8 个 handle (react-moveable)
- 拖动标题栏移动, 拖动 handle 缩放
- 双击标题栏隐藏标题栏

#### 右侧属性面板 (320px, 玻璃)
- 顶部 Tab: 属性 / 主题 / 背景 / 动画 / 项目
- 属性: 标题 / 位置 / 尺寸 / 层级 / 锁定 / 样式 (圆角 / 透明度 / 阴影)
- 主题: 5 种预设 (玻璃 / 亚克力 / 液态 / Fluent / 自定义) + 参数 (透明度 / 模糊 / 圆角 / 高光)
- 背景: 3 个 Tab (纯色 / 渐变 / 图片), 渐变支持 Linear / Radial / Conic, 6 个预设调色板
- 动画: open / close / loop 三槽, 每槽配置 type / duration / delay / easing
- 项目: 复制访问端 URL / 发布 / 导出 JSON / 项目包 / 导入

#### 状态栏 (28px)
- 左: 画布尺寸 / 窗口数 / 选中数
- 右: 保存状态 / 版本

### 2.4 访问端 (`/view/:projectId`)

- 全屏只读画布, 1:1 渲染编辑器所见
- 右上角: 实时同步状态 (绿点 live / 红点 offline) + 项目名
- 媒体自动播放 (静音 / 视频静音自动播放)

### 2.5 预览抽屉 (编辑器内)
- 居中悬浮, 80vh, 70% 缩放预览
- 关闭按钮

### 2.6 JSON 编辑器 (编辑器内)
- 720px 宽, textarea 内嵌 JSON
- 解析失败显示行级错误
- 导入: 追加到画布
